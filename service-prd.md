# service-prd.md

## 한 줄 정의
**회사 업무를 공유할 수 있는 업무자동화 ERP**

---

## Phase 1 — MVP (지출/보고서 + 일정 관리)

### 핵심 기능

#### 1. 지출/보고서 관리
- 보고서 생성 (지출결의서, 주간업무보고, 출장보고서)
- 양식 자동 제공 (필드 자동 채우기)
- 상태 관리 (작성 중 → 제출 → 승인/반려)
- 목록 조회 및 상세 보기

#### 2. 출장/외근 일정 관리
- 일정 등록 (출장, 외근)
- 팀 현황판 (누가 어디에 있는지 표시)
- 캘린더 뷰
- 예상 복귀 시간 표시

### 페이지 구성

```
/                    → 대시보드 (팀 현황판)
/reports             → 보고서 목록
/reports/new         → 보고서 작성
/reports/:id         → 보고서 상세
/schedules           → 일정 관리
```

### 데이터베이스

**테이블:**
- `users` - 팀원 정보
- `reports` - 보고서 (지출/주간/출장)
- `schedules` - 출장/외근 일정

### 성공 기준

✅ 보고서를 작성하고 제출할 수 있다
✅ 팀 현황판에서 팀원의 출장/외근 상황을 실시간으로 볼 수 있다
✅ 보고서 목록에서 상태(작성 중/제출/승인)를 구분할 수 있다

## 기술 스택

- **프론트엔드**: Next.js 16 + TypeScript + Tailwind CSS
- **백엔드**: Supabase
- **인증**: Supabase Auth (향후)

## 주요 고려사항

1. **사용 편의성**: 비개발자도 쉽게 사용
2. **속도**: 빠른 응답 (1초 이내)
3. **정확성**: 데이터 입력 검증
4. **모바일 대응**: 반응형 디자인
