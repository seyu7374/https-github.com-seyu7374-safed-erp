# 업무자동화 ERP 도구

## 한 줄 정의
**회사 업무를 공유할 수 있는 업무자동화 ERP**

---

## 사용 주체
- 우리 회사 (회사 전체 / 팀 단위)

---

## 현재 상황 (문제점)

### 사용 중인 도구
- 엑셀 (데이터 관리)
- 카톡 (소통)
- 수동 처리 (프로세스)

### 주요 불편함
1. **시간 낭비**: 반복적인 수동 작업으로 인한 시간 소모
2. **반복 작업**: 동일한 업무 프로세스를 계속 반복
3. **비효율성**: 도구 간 연동 부재, 정보 산재

---

## 목표 상태 (달성할 것)

### 핵심 개선 사항
1. **실시간 공유**: 팀원 간 정보 즉시 동기화
2. **자동화**: 반복 작업 자동 처리

---

## 핵심 기능 (MVP)

### 1. 지출/보고서 관리

#### 현재 문제점 (엑셀 + 카톡)
- 지출결의서: 엑셀 작성 → 메일/카톡으로 결재 요청 → 수동 승인
- 주간업무보고: 매주 엑셀 작성 → 관리자 확인 (시간 소모)
- 출장보고서: 출장 후 엑셀 작성 → 별도 제출 프로세스

#### 플로우
```
팀원 → 보고서 유형 선택 → 양식 자동 로드 → 작성/제출 → 실시간 결재 → 자동 저장/통계
```

#### 상세 기능
- **보고서 유형별 관리**
  - 지출결의서 (영수증 첨부, 항목별 금액)
  - 주간업무보고 (주간 진행 사항, 다음주 계획)
  - 출장보고서 (출장지, 기간, 비용, 성과)
  - 커스텀 양식 추가 가능

- **스마트 양식**
  - 양식 템플릿 자동 제공 (반복 제출 불필요)
  - 이전 보고서 기반 자동 입력 (중복 정보)
  - 필수 항목 자동 검증

- **실시간 결재**
  - 제출 즉시 관리자에게 알림
  - 결재 진행 상황 실시간 추적
  - 반려 시 자동 수정 요청 및 재제출

- **통계/분석**
  - 팀원별 주간 보고서 자동 수집
  - 지출 내역 자동 집계
  - 출장 현황 대시보드

#### 자동화 항목
- ✅ 양식 템플릿 자동 로드
- ✅ 중복 정보 자동 입력
- ✅ 결재 알림 자동 발송
- ✅ 통계 자동 생성

---

### 2. 출장/외근 일정 관리

#### 현재 문제점 (카톡 + 수동)
- 출장 일정: 카톡으로 공유 → 누가 언제 나가는지 불명확
- 외근 업무: 수동으로 기록 → 팀에서 실시간 파악 어려움
- 누가 사무실에 있는지 확인 불가 → 협업 지연

#### 플로우
```
팀원 → 출장/외근 등록 → 팀 캘린더 실시간 동기화 → 팀 현황판 업데이트 → 알림 발송
```

#### 상세 기능
- **출장 일정 관리**
  - 출장지, 기간, 목적 입력
  - 팀 캘린더에 자동 표시 (색상: 빨강 = 출장 중)
  - 출장 중 업무 위임 설정
  - 출장 보고서 자동 연동

- **외근 업무 현황**
  - 외근 장소, 시간 입력
  - 실시간 팀 현황판 (누가 언제 돌아오는지 표시)
  - 예정된 미팅 시간 자동 반영

- **실시간 팀 현황**
  - 대시보드에서 팀원별 상태 한눈에 보기
  - 사무실 재근/외근/출장 구분
  - 사무실 예상 복귀 시간 표시

#### 자동화 항목
- ✅ 출장/외근 캘린더 자동 동기화
- ✅ 팀 현황판 실시간 업데이트
- ✅ 예정된 미팅 자동 반영
- ✅ 출장 보고서 자동 연동

---

### 3. 프로젝트 관리

#### 플로우
```
PM → 프로젝트 생성 → 단계별 작업 할당 → 진행률 추적 → 산출물 관리 → 완료 & 평가
```

#### 상세 기능
- **프로젝트 구조**
  - 프로젝트 정보 (명칭, 설명, 기간, PM)
  - 단계 설정 (기획 → 개발 → 테스트 → 배포)
  - 각 단계별 작업 할당

- **진행률 추적**
  - 전체 진행률 자동 계산 (하위 작업 기반)
  - 단계별 진행률 시각화
  - 병목 작업 자동 감지

- **산출물 관리**
  - 단계별 산출물 등록
  - 파일 첨부 (설계서, 코드, 테스트 결과 등)
  - 버전 관리 (변경 이력 자동 기록)
  - 검수 승인 플로우

- **팀 협업**
  - 프로젝트별 채팅방
  - 작업 의존도 표시 (A 작업 완료 후 B 작업 시작)
  - 담당자별 역할 명확히

#### 자동화 항목
- ✅ 진행률 자동 계산
- ✅ 지체된 작업 알림
- ✅ 병목 작업 감지
- ✅ 산출물 변경 이력 자동 기록

---

## 설계 원칙

- **검증된 운영 방식 기반**: 새로운 시스템이 아닌, 이미 회사에서 작동하는 방식을 디지털화
- **사용자 중심**: 복잡하지 않게, 직관적인 UI/UX
- **확장 가능**: 향후 추가 기능 수용 가능한 구조
